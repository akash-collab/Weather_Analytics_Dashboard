# Weather Analytics Dashboard

A weather analytics dashboard built with React and Redux Toolkit.

## Features
- Real-time weather data
- 5-day and hourly forecasts
- Search with autocomplete
- Favorites with persistence
- Unit toggle (°C / °F)
- Cached API responses (60s)
- Auto refresh after cache expiry
- Interactive charts (Recharts)

## Tech Stack
- React + Vite
- Redux Toolkit
- Tailwind CSS
- OpenWeatherMap API

## Setup
1. Clone repo
2. Run `npm install`
3. Create `.env` file:
4. VITE_WEATHER_API_KEY=your_api_key
5. Run `npm run dev`
